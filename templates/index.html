Stocks Market Api

Here is an example of how to use the API

                curl -X POST https://aenima-sma.herokuapp.com/api/signup -H 'Content-Type: application/json' \
                -d '{"email":"user@email.com","name":"name", "lastname": "lastname"}' # {"token":"ABC"}
                curl -X POST -H "Authorization: Token ABC" https://aenima-sma.herokuapp.com/api/stocks/AAPL
                    # {"symbol":"AAPL","open":142.99,"lower":142.1201,"higher":146.45,"close_variation":0.37}
